# Getting Started

## Feature Summary

The important features of Essential Report Writer are:

* Support for rendering reports as PDF, Word, and Excel documents
* Support for the following report items: 
* Tablix
* Chart
* Gauge
* Text box
* Shape
* Report parameter
* Data source
