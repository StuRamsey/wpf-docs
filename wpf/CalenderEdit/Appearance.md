# Appearance 

The appearance of the CalendarEdit control can be customized by using the VisualStyle property. The following are the various built-in visual styles for CalendarEditcontrol.

* Blend
* Office2003
* Office2007Blue
* Office2007Black
* Office2007Silver
* ShinyBlue
* ShinyRed
* SyncOrange
* VS2010
* Metro
* Transparent
