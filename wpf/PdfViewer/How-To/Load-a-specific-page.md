# Load a specific page

Navigation to a specific page, through code, is possible using GoToPageAtIndex method.

{% highlight c# %}

pdfViewer1.GoToPageAtIndex(2);

{% endhighlight %}

{% highlight vbnet %}

pdfViewer1.GoToPageAtIndex(2)



{% endhighlight %}
